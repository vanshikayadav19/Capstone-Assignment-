Step 1: Create Topology
Open Cisco Packet Tracer
Drag devices:
2 PCs (PC1, PC2)
1 Switch (2960)
Connect:
PC1 → Switch (Copper Straight Cable)
PC2 → Switch (Copper Straight Cable)
 
  Step 2: Configure IP Addresses
Click PC1 → Desktop → IP Configuration
IP Address: 192.168.1.10
Subnet Mask: 255.255.255.0
Click PC2 → Desktop → IP Configuration
IP Address: 192.168.1.20
Subnet Mask: 255.255.255.0

 Step 3: Switch to Simulation Mode
Click Simulation Mode (bottom right)

 Step 4: Perform Ping
Click PC1 → Desktop → Command Prompt

Type:

ping 192.168.1.20
